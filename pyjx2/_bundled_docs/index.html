    Bienvenidos a PYJX2

    PYJX2 es una herramienta de automatización para Jira y Xray, construida sobre arquitectura limpia y moderna en Python. Está diseñada con el propósito fundamental de agilizar la configuración de pruebas y la recolección de evidencias durante los distintos ciclos del desarrollo de software.

    Quick Start

    Para comenzar de inmediato, instala la herramienta y lanza la interfaz: 

    pip install .
pyjx2 tui

    ¿Qué puedes hacer con PYJX2?

    • Flujo de Preparación (Setup) --- Crea entornos de pruebas completos desde Test Plans. Organiza Test Sets y vincula Test Executions automáticamente.

    • Flujo de Sincronización (Sync) --- Sincroniza evidencias físicas de forma masiva. Identifica archivos, empareja llaves e inyecta adjuntos en Xray sin errores.

    Ecosistemas de Interacción

    • Línea de Comandos (CLI) --- Potente interfaz basada en Typer. Integración nativa en Pipelines CI/CD y scripts de automatización masiva.

    • Interfaz Textual (TUI) --- Experiencia gráfica terminal con Textual. Ideal para operaciones manuales asistidas con feedback visual en tiempo real.

    • API Python Nativa --- Arquitectura limpia y extensible. Usa el patrón Facade para integrar capacidades de PyJX2 en tus propios proyectos Python.
